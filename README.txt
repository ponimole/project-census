This python script uses a dataset of approximately 49000 adult income examples with attributes of education, marital
status, occupation, etc and creates a neural network model that predicts whether or not a random adult's income 
exceeds $50K/yr.

This repository includes:
adult.csv - (training dataset)
adult_test.csv - (test dataset)
census1994.ipynb - (google colab)
census1994 training session.pdf - (training session that achieves 82% accuracy)
README.txt - (read me)
